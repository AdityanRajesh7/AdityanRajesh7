# hi, I'm Adityan 👋

I break things with code, fix them with data, and call it research.  
Love to explore the behavior of learning systems — how they optimize, generalize, and sometimes fail in interesting manners.

---

## what I work on

- **Reinforcement Learning** — building agents from scratch, understanding policy dynamics
- **Probabilistic ML** — uncertainty, inference, and learning under noise
- **Neural Architectures** — experimenting beyond standard models (chaotic dynamics, diffusion, etc.)
- **Optimization** — from gradient descent to bio-inspired algorithms
